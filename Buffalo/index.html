Hunyuan3D-Buffalo 1.0 Pipeline

Hunyuan3D-Buffalo 1.0 is a unified 3D multimodal framework for 3D understanding, text-to-3D generation, and instruction-guided 3D editing. It supports diverse tasks within a single pipeline by connecting language, 3D representations, and generative 3D modules.

The framework uses a shared Hunyuan3D-VLM backbone to bridge 3D QA, grounding, generation, and editing. Based on this unified representation, Hunyuan3D DiT modules further enable scalable multimodal generation and high-quality 3D editing.
